# Plan: Intro Flow Fix, Demo Mode, and Registration Error Handling

## Issues Identified

### 1. Intro Flow Auto-Login
**Problem:** Intro currently auto-logs into demo and redirects to dashboard when it ends.
**Fix:** Intro should only dismiss and show home page. Demo login should be triggered separately by a button click.

### 2. Demo Mode Implementation
**Problem:** Demo button exists in MarketingHero but needs proper demo mode handling.
**Fix:** 
- After demo login, show locked demo mode indicator in dashboard
- Demo account uses existing `SEED_DEMO_ADMIN_EMAIL` / `NEXT_PUBLIC_DEMO_PASSWORD` credentials

### 3. Registration Error Display
**Problem:** Registration errors show generic API error messages.
**Fix:** Registration errors should show "Request sent to super admin" message instead of API errors.

### 4. Features Nav Redirect
**Problem:** "Features" nav item points to `#product` but should go to `#orbit`.
**Fix:** Change Features href from `#product` to `#orbit` in SiteNav.tsx.
