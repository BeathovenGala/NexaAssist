# Plan: Home Page Explore Feature, Demo Mode, and Robot Integration

## Issues Identified

### 1. Missing "Explore Product" Button
**Problem:** No prominent CTA for users to explore product features on home page
- Need a "Explore Product" button that navigates through feature sections
- Should scroll to features/orbit section with smooth animation

### 2. Slow Button Response
**Problem:** General UI sluggishness - buttons require multiple clicks or are slow
- `AnimatedGroup` wraps buttons creating extra DOM layers with framer-motion
- The transition delays may be causing perceived slowness
- `GlowCard` has `pointer-events-none` styles that interfere

### 3. Auth Modal Centering
**Problem:** Sign-in/sign-up cards not centered properly
- `AuthModalHost.tsx` z-index is `z-[100]` - may conflict with other layers
- `SignInCard` wrapper needs explicit centering and pointer-events

### 4. No Demo Mode
**Problem:** Users cannot easily try the product without signing up
- Need "Try Demo Now" button on home page
- Should create auto-login to a demo account (read-only mode)
- Demo data should be visual only (not persisted)

### 5. Missing Dashboard Robot
**Problem:** Robot visible on home page should also appear in dashboard
- Use `InteractiveRobotSpline` or `AnimatedRobotFallback` from chatbot
- Place in dashboard header area
